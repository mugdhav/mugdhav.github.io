Pre-Event Instructions

Greetings, fellow explorer!

Thanks for registering for the event Pie & AI: Pune- Exploring Indian AI.

To get the most from this event, please perform these quick setup steps before the event:

Pre-Event Setup Steps

  1. Create a CoRover.ai BharatGPT account here: https://builder.corover.ai/#/auth/register. You can use your existing Gmail ID for this.
    Already have a CoRover.ai BharatGPT account? Then log into it and await instructions.
  2. Create an account for Sarvam AI here: https://dashboard.sarvam.ai/signin. Use your existing Gmail or Microsoft ID for this.
    Already have a Sarvam AI account? Then log into it and await instructions.
  3. Optionally, if you are a developer interested in exploring Indian LLM APIs, then:
    1. Log into your Sarvam AI account and click Get API Key on Sarvam AI Dashboard. Save the API Key.
    2. Create your Google Colab account here: https://colab.research.google.com. Use your existing Gmail ID for this.
    Skip these steps, if you are not interested development with Indian LLM APIs.

That's it! You are all set for the event.
