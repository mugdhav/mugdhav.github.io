Pre-Event Setup Instructions

Greetings, fellow AI explorer!

Thanks for registering for the event Pie & AI: Pune - Extracting and Structuring Information.

To get the most from this event, please follow these steps before the event:

Setup Steps

1 LandingAI Agentic Document Extraction (ADE)

1 Create a LandingAI account, navigate to Agentic Document Extraction, and generate an API key. Save your API key securely for use during the event for hands-on activities.

3 Explore the LandingAI ADE API documentation to understand information extraction.

2 Google AI Studio and Document AI

1 Access Google AI Studio with your Gmail ID and get an API key.

2 Optionally, access Google Colab and add your API key as a secret. Skip this step if using another IDE.

3 Explore Google Document AI documentation. Also, review the example code in this Google Colab notebook: PDF Structured Outputs on Invoices and Forms.

3 DocETL Framework

1 Visit the DocETL website and the DocETL GitHub repository to understand the framework.

3 Read the official documentation. Try the DocETL Python API Tutorial.

4 Follow the Quick Start Tutorial to set up a DocETL API key using your API key created in Google AI Studio.

Need Help?

In Google AI Studio Chat, ask gemini-2.5-flash-lite to provide you step-by-step instructions.

That's it! You are all set for the event.
