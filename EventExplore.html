Generate Content with Locally-Run Open Source Models

Running Large Language Models (LLMs) locally gives you complete privacy, eliminates internet dependency, reduces costs, and provides full control over your AI interactions. Your data never leaves your device, making it ideal for sensitive work or areas with limited connectivity.

Popular tools for running LLMs locally include Ollama, LM Studio, and GPT4All. For this event, we'll be using Ollama to run models locally.

Step-by-Step Instructions:

1. Run phi3:3.8b using Desktop App:
  1. Open the Ollama desktop application on your computer
  2. In the search bar, type phi3:3.8b
  3. Click on the phi3:3.8b model to download and install it
  4. Once downloaded, click "Chat" to start interacting with the model
  5. Test the model by asking: "What are the benefits of running AI models locally?"
2. Run gemma2:2b from Command Prompt:
  1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
  2. Type the command: ollama run gemma2:2b
  3. Wait for the model to download and initialize (this may take a few minutes)
  4. Once the prompt appears, start chatting with the model
  5. Test with: "Explain quantum computing in simple terms"
  6. Type /bye to exit the chat

Note: Make sure you have sufficient disk space (at least 4GB free) and a stable internet connection for the initial model downloads.

Generate Content Using Online Open Source Models

Important: Before following these steps, follow the set-up instructions.

  1. Log into your Google Colab account. In Secrets pane , add a new secret with Name sarvam. As the secret's Value, enter the API key you created from Sarvam dashboard. Enable the Notebook access to the secret.
  2. Download the sarvam_translator.ipynb file (notebook). Upload this notebook in your Google Colab from File > Upload Notebook menu.
  3. Explore the code in the notebook.
    The notebook contains a Python program that can translate your input in any Indian language to another Indian language of your choice using the Sarvam API. See Sarvam API documentation.
  4. Use Google Input Tools to prepare text of your choice in any Indian language.
  5. Run the notebook. When prompted for input, enter the text you prepared using Google Input Tools.
