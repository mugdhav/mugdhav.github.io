Workshop Instructions

Prerequisites

Please complete the setup before following these instructions.

Message from Andrew Ng

Watch Andrew Ng's special message for the Pie & AI: Pune event participants.

Download Sample Files

Download the PieAIEvent3ExtractingInformation_Input.zip file and extract it on your local drive. The resulting folder contains sample documents and images to extract information from.

LandingAI Agentic Document Extraction (ADE)

Refer to LandingAI Quick Start and Parsing Basics examples, and write your own code to parse the sample files. Generate parsed output as JSON files.

Google Document AI

Follow the Google Document AI codelab, providing the downloaded sample files as input. Generate parsed output as JSON files.

DocETL Framework

Follow the DocETL Quick Start Tutorial - Python API, providing previously created JSONs as input.

Ready to Extract?

These hands-on activities will give you practical experience with different information extraction approaches. Come prepared with your API keys and get ready to process, extract, and structure data from various document types!
